\doxysubsection{Detailed Description}
The purpose of this class is to represent the \mbox{\hyperlink{class_transportation___management___system_1_1_planner}{Planner}} \mbox{\hyperlink{class_transportation___management___system_1_1_user}{User}}. 

This class represents the role of a \mbox{\hyperlink{class_transportation___management___system_1_1_planner}{Planner}} \mbox{\hyperlink{class_transportation___management___system_1_1_user}{User}}, which represents a \mbox{\hyperlink{class_transportation___management___system_1_1_user}{User}}, who is responsible for planning for an order and completing an order. Additionally, a \mbox{\hyperlink{class_transportation___management___system_1_1_planner}{Planner}} can also generate a summary report of all invoice data for either all time or the past 2 weeks\textquotesingle{} of simulated time.
