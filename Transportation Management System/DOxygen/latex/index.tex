The {\bfseries{TMS}} application is a prototype for the {\itshape shipping}, and {\itshape logistics} industry. It allows for the users to manage the purchasing, schedulling, monitoring and billing of multiple organizations that close a contract to have their order delivered by a partner carrier, whether it is via train or truck.

The application is written mainly in C\#. For more information about this programming language, you can visit\+:
