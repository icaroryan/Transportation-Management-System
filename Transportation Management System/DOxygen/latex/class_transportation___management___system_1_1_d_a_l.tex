\doxysubsection{Detailed Description}
The purpose of this class is to act as the Data Access Layer for the system. 

This class will manage all the communication between the controller to the database. Therefore, each query or change to the database need to be managed by the \mbox{\hyperlink{class_transportation___management___system_1_1_d_a_l}{DAL}} class.
