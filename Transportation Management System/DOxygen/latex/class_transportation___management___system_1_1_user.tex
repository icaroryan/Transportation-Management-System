\doxysubsection{Detailed Description}
The purpose of this class is to represent the role of a user. 

This class represents the role of a user when they login to the TMS application. There are three different types of users that will inherit from \mbox{\hyperlink{class_transportation___management___system_1_1_user}{User}}; \mbox{\hyperlink{class_transportation___management___system_1_1_buyer}{Buyer}}, \mbox{\hyperlink{class_transportation___management___system_1_1_planner}{Planner}}, \mbox{\hyperlink{class_transportation___management___system_1_1_admin}{Admin}}.
